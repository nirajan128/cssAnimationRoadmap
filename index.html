CSS ANIMATION

Transition

The first property is Transition propertey.CSS transitions allows you to change property values smoothly, over a given duration

TRANSITION SEVERAL
You can change several property values for different time. this expample when hovered the color property takes 2s to change while font size takes only 1s

TRANSITION ALL PROPERTY
You can also change all properties at same time using all and defining the duration

DELAY PROPERTY
You can add delay property to cerntain time,after which the transition starts

TRANSITION PROPERTY AND DURATION
When you use transition-property also make sure to use transition duration which will define the duration of the transition

TRANSITION TIMING PROPERTY: EASE
specifies a transition effect with a slow start, then fast, then end slowly (this is default)

TRANSITION TIMING PROPERTY: LINEAR
specifies a transition effect with the same speed from start to end

TRANSITION TIMING PROPERTY: EASE-IN
specifies a transition effect slow start

TRANSITION TIMING PROPERTY: EASE-OUT
specifies a transition effect slow end

TRANSITION TIMING PROPERTY: EASE-IN-OUT
specifies a transition effect with slow start and end

SHORT HAND PROPERTY
Instead of wirting each transition property, we can use a short hand method
transition: transition-property duration timing-function delay
In this Button, the following proerties have been used in short hand ,
transition: width 0.5s ease-in-out 0.7s;

Animation

In this section we will be learning about keyframes and animation properties of css

From to: You can use this properto to define a starting point and a end point

@keyframe animationName {
from{}
to{}
}

% : You can also define different animations for different % for an element

@keyframe animationName{
10%{ }
40%{ }
80%{ }
100%{ }
}

Animation-name:

The name of the thransition defined in the @keyframe

Animation-duration:

Sets the duration for the animation. animation-duration:4s which means it will take 4 second to complete the animation.

Animation-delay:

Sets the delay to start the animation

Animation-iteration:

Set the iteration for animation values:

  • Infinite: animation runs infinite number of times

    • Animation-timing-function:

    Sets the timing function for animation
    values:

  • Ease
  • Ease-in
  • Ease-out
  • Ease-in-out
  • linear

    • Animation-direction:

    Determines whether the animation should play forwards, backwards, or alternate between forwards and backwards on each iteration.

    Animation-fill-mode:

    Defines what styles are applied to the element before and after the animation. values include:

  • none
  • forward
  • Backward

    • @keyframes grow{
    from{width: 80px; height: 100px;}
    to{width: 320px; opacity: 100%; background-color: rebeccapurple;}
    }
     .box{
    animation-name: grow;,

    Short-hand property

    .box{
    animation: name duration timing-function delay iteration-count direction fill-mode;
    }
    .box{
    animation: growShrink 3s ease-in none infinite alternate ;
    }

    Transform Property

    Applies 2D or 3D transformation to an element.

    Translate-Y: translates element vertically

    Translate-X: translates element horizontally

    Translate(X,Y): Where X value is position right and Y value is position down.

    ScaleY: Scales the element vertically

    ScaleX: Scales elemnts horizontally

    Scale(X,Y): Where X takes horizontal value and Y takes vertical value

    Rotate: Rotates the element, does not have X and Y property.

    SkewY: Skews the element vertically

    SkewX: Skews the element horizontally
